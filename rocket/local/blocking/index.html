Blocking local dispatching of requests.

This module contains the blocking variant of the local API: it can be used in Rust’s synchronous #[test] harness. This is accomplished by starting and running an interal asynchronous Runtime as needed. For the asynchronous variant, see asynchronous.

See the top-level documentation for more usage details.
