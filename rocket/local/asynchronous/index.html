Module rocket::local::asynchronous

Asynchronous local dispatching of requests.

This module contains the asynchronous variant of the local API: it can be used with #[rocket::async_test] or another asynchronous test harness. For the blocking variant, see blocking.

See the top-level documentation for more usage details.
